Click Install on the scripts for your role. Tampermonkey will handle the rest.

First time? Make sure you have Tampermonkey installed in your browser first. If you already have an old version of a script, delete it from your Tampermonkey dashboard after installing the new one.

TECH — Technicians

Optional scripts for techs

Auto Add Labor + Tech Time Panel Automatically adds the Labour service line to every work order and sets it up correctly. Also adds a floating panel to log tech hours directly from the order page without opening the Job.
▶ Install
Auto Grow Work Order Description The Work Order Description text box automatically grows as you type so you can see the full text without scrolling inside the box.
▶ Install
Force Contact Section Collapsed Keeps the Contact section collapsed by default on the order page. Techs rarely need it — this keeps it out of the way.
▶ Install
Hide Totals Footer Hides the cost totals bar at the bottom of the order page. Techs don't set pricing, so this removes the distraction.
▶ Install
Orders Grid Filter Optimizer Makes the Customer Orders list much faster by loading all orders locally. Adds WIP filter, Clear Filters, and Order Status dropdown. Completed and Cancelled orders are shown by default.
▶ Install
Parts Preloader Loads and caches the full parts list in the background so searching for parts is instant. Refreshes every hour automatically — use the Refresh Parts button to force an update.
▶ Install
Time Expanded Section Trimmed Adds a Time Expanded section directly below the parts list on the order page. Shows the job's service lines (hours logged) without needing to open the Job separately. Clutter removed.
▶ Install
Uppercase Forced Work Order Description Automatically converts the Work Order Description, Customer Snag, and Internal Snag to uppercase when you save. No need to type in caps.
▶ Install

▶ Install
